# GameboyTetris
Faithful Gameboy Emulator project aimed at recreating the Gameboy Tetris Experience. WIP as of right now.

Using Simple Timing for now (Only ensuring registers and flags for each cycle is correct). Might refactor into detailed timing in the future.
